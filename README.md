# unity-backend-study

본 레포지토리는 유튜브 강의 **「뒤끝(Backend)을 이용한 온라인 게임 제작」**을 수강하며  
Unity 기반 온라인 게임 개발과 Backend 연동을 학습한 내용을 정리한 개인 학습 프로젝트입니다.

---

## 📌 프로젝트 목적

- Unity를 활용한 온라인 게임 개발 흐름 이해
- Backend(뒤끝)를 이용한 서버 연동 구조 학습
- 로그인, 데이터 저장, 랭킹 등 온라인 게임의 기본 기능 구현
- 학습 과정 및 코드 변경 이력 기록

---

## 🛠 사용 기술

- **Game Engine**
  - Unity

- **Backend**
  - 뒤끝

- **Language**
  - C#
